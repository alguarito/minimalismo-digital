# El Plano del Minimalismo Digital

**El Plano del Minimalismo Digital** es una Single Page Application (SPA) diseñada como portal educativo y laboratorio técnico para la implementación de Inteligencia Artificial en entornos docentes. 

Desarrollada bajo la dirección del **PhD. Álvaro Cárdenas Orozco**, esta plataforma busca el "Rescate Pedagógico" mediante la reducción del ruido digital y la construcción de infraestructuras invisibles pero potentes para el docente.

## 🚀 Características Principales

- **Navegación SPA**: Interfaz fluida basada en el concepto de plano arquitectónico.
- **Laboratorio de IA (M1)**: Generador de prompts encadenados para la planeación curricular basado en la normatividad del MEN (Colombia).
- **Sincronización Técnica**: Metodología paso a paso para la creación de Agentes IA (TecnoSapiens) con salida en LaTeX profesional.
- **Estética Minimalista**: Diseño técnico en escala de grises con tipografía jerárquica y limpia.

## 🛠️ Tecnologías Utilizadas

- **Core**: HTML5, Vanilla JavaScript.
- **Estilos**: Tailwind CSS (v3), CSS3 Custom Animations.
- **Iconografía**: Lucide Icons.
- **Tipografía**: Google Fonts (Inter, Roboto Mono).

## 📂 Estructura del Proyecto

- `index.html`: Estructura principal, secciones de contenido y laboratorios.
- `app.js`: Lógica de navegación, generación de prompts y utilidades.
- `style.css`: Estilos personalizados, cuadrícula de plano y notificaciones.
